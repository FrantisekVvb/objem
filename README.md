# Objem

Interaktivní aplikace pro sestavování kvádru z krychlí a výpočet objemu.

## Spuštění online

Aplikace běží na GitHub Pages:

**https://frantisekvvb.github.io/objem/**

## Spuštění lokálně

```bash
git clone https://github.com/FrantisekVvb/objem.git
cd objem
npm start
```

Potom otevřete v prohlížeči **http://localhost:3471**.

## Požadavky

- Node.js 18 nebo novější
